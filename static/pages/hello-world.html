Javascript by Example: Hello World!

For our first program, we will create a simple "hello world" program. Don't worry if some of the syntax is unfamilar, we will explain the concepts along the way.

If you are familar with other programming languages, console.log is similar to standard output. console.log prints whatever message you pass along to your web console (right click on your browser and select Inspect. There should be a "Console" tab, where the browser prints messages for the Javascript it executes).
