Javascript by Example: Data Types

There are 7 primitive data types: string, number, bigint, boolean, undefined, symbol and null. In JS, a primitive is data that is not an object and does not contain any methods. All primitives are immutable (in other words, they cannot be changed).

Strings are sequences of characters. JS supports single quote strings (''), double quote strings (""), and template strings using backticks (``).

Number, in JS, is a numeric data type that is double-precision 64-bit floating point.

BigInt is the other numeric primitive available in JS. It is used primarily to represent whole numbers larger than 2^53 - 1 (the maximum safe integer available from Number). They are created using the BigInt constructor or appending n to a number literal.

Boolean is a logical value that can only be one of two values: true or false

Undefined is a value that is automatically assigned to variables or arguments of a function when no actual argument is passed.

A Symbol is created using the factory function Symbol(). The function takes an optional string, and returns a new symbol. Symbols are dynamically produced and are anonymous, unique values. No two symbols are equal.

Null, in JS, represents the intentional absence of a value. In other words, null is explictly assigned to represent no value, while undefined is implicitly assigned and represents that there has been no assignment in value.
