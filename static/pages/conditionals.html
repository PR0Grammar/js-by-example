Javascript by Example: Conditionals (If/Else)

Conditionals in JS are fairly straightforward, with some caveats.

if statements in JS execute a specific block of code if a condition is truthy. Truthy values in JS are values that are considered true in the context of booleans. Truthy values in JS are all values that are not considered falsy. The falsy values are: false, 0, -0, 0n, "", null, undefined, and NaN.

JS supports else if statements. else if statements execute a piece of code if the condition is truthy AND if the top level if statement and previous else if statements (if there are any) are evaluated to false. You can view it similar to a switch statement, where you have a chain of conditions that you are checking (if -> else if -> else if -> else if...).

else statements include a block of code that is executed if the top level if statement evaluates to false AND any else if statements that follow the top level if statement but occur before the else statement. In other words, if all the previous statements in the conditional chain evaluate to false (if -> else if .... -> else), then only the code in the else block is executed.
